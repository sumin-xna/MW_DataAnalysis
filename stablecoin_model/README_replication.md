# Replication Instructions

## Overview
This project studies stablecoins and parallel exchange rates using a global-games framework.

## Running the model
1. Run the full solver:
   python full_solver.py

2. Outputs:
   - figures/
   - summary results
   - calibration parameters

## Key Results
- Stablecoin demand increases with overvaluation
- Stablecoins increase crisis risk
- Welfare is non-monotonic

## Notes
The fast solver is used for calibration. The full solver corresponds to the paper.
